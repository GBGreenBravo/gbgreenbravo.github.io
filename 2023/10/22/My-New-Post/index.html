첫 포스트입니다!

hexo를 활용한 정적 배포입니다.

깃허브 연동 시 제가 부딪혔던 난관 2개를 소개하고자 합니다.

hexo-deployer-git

hexo-deployer-git 플러그인 설치 시 눈을 부릅 뜨시길 바랍니다.
hexo-deployer-git을 hexo-developer-git로 보고 헤매지 않으시길 기원하겠습니다.

Github Repository

Github Repository 생성 시, 레포 이름을 깃허브 닉네임으로 설정하셔야 합니다.
다른 이름으로 도전하신다 하더라도, 깃허브 페이지에서 “There isn’t a GitHub Pages site here.”라는 오류 메시지와 친분을 쌓을 절호의 기회를 갖게 되신 겁니다;)
